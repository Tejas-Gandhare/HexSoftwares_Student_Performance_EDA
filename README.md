# Student Performance EDA 📊

This project performs Exploratory Data Analysis (EDA) on a publicly available Student Performance dataset from Kaggle.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Analysis Performed
- Data cleaning
- Missing value handling
- Statistical summary
- Histogram plots
- Correlation heatmap
- Boxplot for outliers

## Dataset
Student Performance Dataset (Kaggle)

## Key Insights
- Reading and writing scores are highly correlated
- Most scores lie in the 60–80 range
- Some math score outliers are present
